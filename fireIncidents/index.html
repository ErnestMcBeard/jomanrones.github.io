Baton Rouge Fire Incidents

Purpose

This site shows fire incident data reported by Baton Rouge Fire Departments. Details about each incident can be shown by clicking on the icon for the incident. Some incidents from places outside the Baton Rouge limits were not provided like in Central or Zachary. Using this map, we can see where the most incidents occur and can change staffing and funding for different fire departments based on need.

Sources
